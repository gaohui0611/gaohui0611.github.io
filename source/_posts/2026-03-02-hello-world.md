# 欢迎来到我的博客

这是使用 Hexo + GitHub Pages 搭建的个人博客！

## 关于这个博客

- 框架: [Hexo](https://hexo.io/)
- 主题: [NexT](https://github.com/next-theme/hexo-theme-next)
- 托管: [GitHub Pages](https://pages.github.com/)

## 功能特性

✨ 支持 Markdown 语法
🚀 自动部署 - 推送代码即发布
📱 响应式设计 - 支持移动端
🔍 SEO 优化

## 代码示例

```javascript
console.log('Hello, World!');
```

```python
def hello():
    print("Hello, World!")
```

## 快速开始

只需三步：

1. 在 `source/_posts/` 目录下创建 MD 文件
2. `git add .` && `git commit -m "新增文章"`
3. `git push` - 自动触发部署！

开始你的写作之旅吧！
